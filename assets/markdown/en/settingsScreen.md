### Units of Measurement

![Units](resource:assets/markdown/en/settingsScreen_units.png)

Opens a separate screen for choosing units for each type of quantity:

| Quantity | Options |
|---|---|
| Velocity | m/s, fps, km/h, mph |
| Distance | metre, yard, foot |
| Sight height | mm, cm, inch |
| Pressure | hPa, mmHg, inHg, psi |
| Temperature | Celsius, Fahrenheit |
| Drop / Windage | m, cm, mm, inch, foot |
| Drop / Windage angle | mil, MOA, mRad, cm/100m, in/100yd |
| Energy | joule, ft·lb |
| Bullet weight | grain, gram |
| Bullet length | mm, cm, inch |
| Bullet diameter | mm, cm, inch |
| Target size | mil, MOA, mRad, cm/100m, in/100yd |

### Home Screen Settings

![Adjustments](resource:assets/markdown/en/settingsScreen_adjustment.png)

**Adjustment Display** — opens a screen with adjustment appearance settings:

- **Format** — how direction is indicated: ↑/↓ (arrows), +/− (signs), U/D (letters)
- **Adjustment units** — which units to display: mrad · MOA · mil · cm/100m · in/100yd · click

**Show subsonic transition** — highlights the row in the table and chart where the bullet goes subsonic.

**Trajectory table step** and **chart step** — interval between rows/points (in the current distance unit). A smaller value means more detail but slower calculation.

### Backup

- **Export** — saves all profiles, cartridges, sights, and settings to an `.ebcp` file
- **Import** — restores data from an `.ebcp` file

---

### Collection

The collection is a database of weapons and cartridges for quick profile creation. Displays the current version hash and allows checking for updates.
