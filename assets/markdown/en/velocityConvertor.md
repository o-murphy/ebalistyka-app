## Velocity Converter

Converts velocity between m/s, km/h, fps, mph and Mach number.

Enter a value in the top field and select units — all other fields update instantly.

---

### Mach Number and Atmosphere

Mach number depends on the speed of sound, which varies with atmospheric conditions.

By default, the ICAO standard atmosphere is used (15 °C, 1013.25 hPa, altitude 0 m).

**"Custom Atmosphere" toggle** — lets you enter real conditions:

- **Temperature** — air temperature
- **Pressure** — atmospheric pressure
- **Humidity** — relative humidity (%)
- **Altitude** — elevation above sea level

When any of these parameters change, the Mach value is recalculated automatically.
