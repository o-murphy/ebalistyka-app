## Target Distance

Calculates the distance to a target from its physical size and angular size in the scope.

Formula: **D = size / angle** (mil-relation formula).

---

### Input Fields

- **Target size** — the real physical size of the target (mm, cm, m, inch, foot)
- **Angular size** — the apparent size of the target in the scope (mil, MOA, mRad)

The result (metres, yards, feet) updates instantly.
