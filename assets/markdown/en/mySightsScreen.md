## My Scopes

Your personal scope library. Tap the **✓ button** (bottom-right of a card) to assign a scope to the current profile. The currently assigned scope is highlighted and shown first.

---

### Adding a scope

Tap **+** to open the add sheet:

- **Create new** — set up a scope from scratch
- **From collection** — pick a preset from the built-in database
- **Import from file** — load a `.ebcp` file from your device

---

### Scope actions (⋯)

| Action | Description |
|---|---|
| **Edit** | Open the scope wizard to modify its parameters |
| **Duplicate** | Create a copy with a new name |
| **Export** | Save or share as a `.ebcp` file |
| **Delete** | Permanently remove the scope |

---

### Filter

Tap the filter icon (top-right) to narrow the list by **manufacturer** or **focal plane** (FFP / SFP). A badge indicates an active filter.
