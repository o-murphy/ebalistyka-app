## Scope Collection

A built-in database of preset scopes. Tap the **✓ button** (bottom-right of a card) to select it for the current profile.

Each tile shows:
- **Manufacturer** and **model name**
- **Reticle** pattern name
- **Vertical** and **horizontal click** values
- **Sight height** (mounting height above bore)
- **Magnification** range

---

### Search

Type in the search bar to filter by name or manufacturer. The list updates instantly.

---

### Filter

Tap the filter icon (top-right) to narrow the list by:
- **Manufacturer**
- **Focal plane** — First Focal Plane (FFP) or Second Focal Plane (SFP)

A badge on the icon indicates an active filter.
