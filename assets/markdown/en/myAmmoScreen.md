## My Ammo

Your personal cartridge library. Tap the **✓ button** (bottom-right of a card) to assign a cartridge to the current profile. The currently assigned cartridge is highlighted and shown first.

---

### Adding a cartridge

Tap **+** to open the add sheet:

- **Create new** — set up a cartridge from scratch
- **From cartridge collection** — pick a full cartridge preset from the built-in database
- **From bullet collection** — pick a projectile preset and open it in the editor
- **Import from file** — load a `.ebcp` or `.a7p` file from your device

---

### Cartridge actions (⋯)

| Action | Description |
|---|---|
| **Edit** | Open the cartridge wizard to modify its parameters |
| **Duplicate** | Create a copy with a new name |
| **Export** | Save or share as a `.ebcp` file |
| **Delete** | Permanently remove the cartridge |

---

### Filter

Tap the filter icon (top-right) to narrow the list by **caliber**, **manufacturer**, or **projectile weight** range. A badge indicates an active filter.

---

### Caliber mismatch

If the selected cartridge caliber does not match the profile's weapon caliber, a prompt will appear offering to update either the cartridge or the weapon to resolve the mismatch.
