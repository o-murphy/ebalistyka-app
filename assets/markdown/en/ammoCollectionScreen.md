## Cartridge / Bullet Collection

A built-in database of preset cartridges and projectiles. Tap the **✓ button** (bottom-right of a card) to select it for the current profile.

Each tile shows:
- **Manufacturer** and **cartridge name**
- **Projectile name**
- **G1 BC** and **G7 BC** — the primary drag model is highlighted
- **Muzzle velocity**
- **Caliber**, **projectile weight**, and **projectile length**

---

### Search

Type in the search bar to filter by name, manufacturer, or projectile name. The list updates instantly.

---

### Filter

Tap the filter icon (top-right) to narrow the list by:
- **Caliber**
- **Manufacturer**
- **Projectile weight** range

A badge on the icon indicates an active filter.

---

### Caliber pre-filter

When opening the collection from a profile that already has a weapon assigned, the list is automatically pre-filtered to show only matching calibers. You can remove this filter manually in the filter sheet.
