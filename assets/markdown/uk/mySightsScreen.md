## Мої приціли

Особиста бібліотека прицілів. Натисніть кнопку **✓** (внизу праворуч на картці), щоб призначити приціл поточному профілю. Поточний призначений приціл виділено та показано першим.

---

### Додавання прицілу

Натисніть **+** щоб відкрити меню додавання:

- **Створити новий** — налаштувати приціл з нуля
- **З колекції** — обрати пресет із вбудованої бази даних
- **Імпортувати з файлу** — завантажити файл `.ebcp` з пристрою

---

### Дії з прицілом (⋯)

| Дія | Опис |
|---|---|
| **Редагувати** | Відкрити майстер прицілу для зміни параметрів |
| **Дублювати** | Створити копію з новою назвою |
| **Експортувати** | Зберегти або поділитись як файл `.ebcp` |
| **Видалити** | Безповоротно видалити приціл |

---

### Фільтр

Натисніть іконку фільтра (вгорі праворуч) щоб звузити список за **виробником** або **фокальною площиною** (FFP / SFP). Значок з бейджем означає активний фільтр.
