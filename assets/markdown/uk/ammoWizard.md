## Новий набій / Редактор набою

![Огляд редактора набою](resource:assets/markdown/uk/ammoWizard_overview.png)

Створіть або відредагуйте пресет набою. Форма поділена на секції: **Загальне**, **Куля** (включно з моделлю опору), **Набій** (початкова швидкість і чутливість пороху), **Умови пристрілки** та необов'язкова **корекція Коріолісу**.

Натисніть **Зберегти** (нижній правий кут) для підтвердження або **Відхилити** (нижній лівий кут) для скасування без змін.

---

### Загальне

- **Назва** — мітка для ідентифікації набою у власній бібліотеці
- **Виробник** — назва виробника; використовується для фільтрації в бібліотеці набоїв

---

### Куля

![Секція кулі](resource:assets/markdown/uk/ammoWizard_projectile.png)

- **Назва кулі** — модель або довільне позначення кулі
- **Калібр** — фактичний діаметр кулі; береться з пов'язаного профілю зброї та відображається як не редаговане поле
- **Маса** — маса кулі; впливає на розрахунки енергії, віддачі та траєкторії (обов'язково)
- **Довжина** — повна довжина кулі; використовується для розрахунків гіроскопічної стабільності (обов'язково)

---

### Модель опору

![Секція моделі опору](resource:assets/markdown/uk/ammoWizard_dragModel.png)

Оберіть балістичну модель, яка описує гальмування цієї кулі в повітрі:

| Модель | Найкраще підходить для |
|--------|------------------------|
| **G1** | Плоскохвостові, тупоголові або класичні спортивні кулі |
| **G7** | Кулі з човниковим хвостом для дальніх дистанцій — зазвичай точніші для сучасних гвинтівкових куль |
| **CUSTOM** | Будь-яка куля з відомою таблицею Mach → Cd (наприклад, з даних доплерівського радара або виробника) |

**G1 / G7 — Один BC**

Введіть один балістичний коефіцієнт, що застосовується на всіх швидкостях. Простий у налаштуванні; точність може знижуватися в трансзвуковій зоні.

**G1 / G7 — Таблиця мульти-BC**

Увімкніть перемикач **Мульти-BC**, щоб ввести таблицю значень BC для різних діапазонів швидкостей. Застосунок інтерполює між точками, підвищуючи точність у всьому діапазоні швидкостей (надзвук → трансзвук → дозвук). Натисніть **Редагувати таблицю**, щоб відкрити редактор точок перелому.

**Таблиця власного опору (CUSTOM)**

Натисніть **Редагувати таблицю опору**, щоб ввести пари Mach → Cd. Таблиця повинна охоплювати весь очікуваний діапазон швидкостей кулі. Рядки сортуються за Mach за зростанням при збереженні.

---

### Набій

![Секція набою](resource:assets/markdown/uk/ammoWizard_cartridge.png)

- **Початкова швидкість** — виміряна або вказана виробником швидкість на дулі, при еталонній температурі нижче (обов'язково)
- **Температура вимірювання ПШ** — температура навколишнього середовища, при якій виміряно або вказано ПШ; використовується як еталонна точка для корекції чутливості пороху
- **Чутливість пороху** — увімкніть цей перемикач для застосування температурної корекції ПШ; коефіцієнт чутливості встановлюється через таблицю вимірювань (див. нижче)

---

### Чутливість пороху (коли ввімкнено)

![Секція чутливості пороху](resource:assets/markdown/uk/ammoWizard_powderSens.png)

Коли перемикач **Чутливість пороху** увімкнено, з'являється додаткова секція:

- Поточна скоригована ПШ при заданій температурі пороху відображається в реальному часі
- **Окрема температура пороху** — увімкніть, щоб задати окрему температуру пороху (наприклад, набої зберігаються в теплі, а стрільба у холод); коли вимкнено — використовується температура повітря з екрану умов
- **Розрахувати з вимірювань** — натисніть, щоб відкрити таблицю вимірювань T → V; введіть мінімум два виміри температура/швидкість зроблені на стрільбищі; застосунок автоматично розраховує коефіцієнт чутливості (%/15°C)

---

### Умови пристрілки

![Секція умов пристрілки](resource:assets/markdown/uk/ammoWizard_zeroing.png)

Атмосферні умови, зафіксовані під час пристрілки. Застосунок використовує їх для компенсації зміни атмосфери між пристрілкою та поточними умовами стрільби.

- **Дистанція нуля** — відстань, на якій виконано пристрілку
- **Кут огляду** — вертикальний кут до цілі під час пристрілки (0° = рівна поверхня)
- **Температура** — температура повітря в місці пристрілки
- **Тиск** — атмосферний тиск у місці пристрілки
- **Вологість** — відносна вологість повітря в місці пристрілки
- **Висота над рівнем моря** — висота місця пристрілки

---

### Зміщення нуля

Точне налаштування нуля шляхом введення кількості кліків, вже накрученої під час пристрілки. Використовуйте для зсуву балістичного нуля без зміни дистанції пристрілки.

- **Вертикальне зміщення** — кількість кліків (позитивне = вгору)
- **Горизонтальне зміщення** — кількість кліків (позитивне = вправо)
- **Одиниця кліку** — одиниця для значень зміщення; має відповідати одиниці маховика прицілу

---

### Корекція Коріолісу при пристрілці

![Секція Коріолісу](resource:assets/markdown/uk/ammoWizard_coriolis.png)

Увімкніть **ефект Коріолісу**, щоб врахувати обертання Землі під час пристрілки. Актуально на дистанціях понад 800–1000 м.

- **Широта** — географічна широта місця пристрілки (−90° до +90°)
- **Азимут** — напрямок стрільби відносно півночі (0–360°)
