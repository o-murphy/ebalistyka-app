## Новий приціл / Редактор прицілу

Заповніть поля, щоб створити або оновити пресет прицілу у власній бібліотеці.

---

### Загальне

- **Назва** — мітка для ідентифікації прицілу
- **Виробник** — назва виробника (використовується для фільтрації в бібліотеці прицілів)

---

### Монтаж

- **Висота прицілу** — відстань від осі каналу ствола до оптичної осі прицілу; впливає на розрахунок вертикальних поправок
- **Горизонтальне зміщення** — бічне зміщення прицілу від осі ствола; залиште 0 для центрально встановленої оптики

---

### Сітка

- **Тип сітки** — оберіть SVG-зображення сітки для відображення на екрані прицілювання
- **Фокусна площина** — FFP (перша фокусна площина), SFP (друга фокусна площина) або LWIR (тепловізійна)
- **Мін / Макс збільшення** — діапазон кратності прицілу; використовується разом із фокусною площиною для коректного масштабування сітки при будь-якій кратності

---

### Кліки

- **Вертикальний / Горизонтальний клік** — кутове значення одного клацання маховика
- **Одиниця кліку** — одиниця виміру кліків (MOA, MIL, SMOA тощо)
