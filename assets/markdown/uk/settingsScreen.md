### Одиниці вимірювання

![Одиниці](resource:assets/markdown/uk/settingsScreen_units.png)

Відкриває окремий екран з вибором одиниць для кожного типу величин:

| Величина | Варіанти |
|---|---|
| Швидкість | м/с, fps, км/г, mph |
| Дистанція | метр, ярд, фут |

| Тиск | hPa, mmHg, inHg, psi |
| Температура | Цельсій, Фаренгейт |
| Знесення / Вітер | м, см, мм, дюйм, фут |

| Енергія | Дж, ft·lb |

| Довжина кулі | мм, см, дюйм |
| Діаметр кулі | мм, см, дюйм |
| Розмір цілі | mil, MOA, mRad, см/100м, дюйм/100ярд |

### Налаштування головного екрана

![Поправки](resource:assets/markdown/uk/settingsScreen_adjustment.png)

**Відображення поправок** — відкриває екран з налаштуваннями вигляду поправок:

- **Формат** — спосіб позначення напрямку: ↑/↓ (стрілки), +/− (знаки), U/D (літери)
- **Одиниці поправок** — які одиниці відображати: mrad · MOA · mil · см/100м · дюйм/100ярд · клік

**Показувати дозвуковий перехід** — підсвічує рядок у таблиці та графіку де куля переходить у дозвуковий режим.

**Крок таблиці траєкторії** і **крок графіка** — інтервал між рядками/точками (у поточних одиницях дистанції). Менше значення = більше деталей, але повільніший розрахунок.

### Колекція

Колекція — база зброї та набоїв для швидкого створення профілів. Відображає поточний хеш версії та дозволяє перевірити наявність оновлень.
